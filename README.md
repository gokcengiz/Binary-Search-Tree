# Binary-Search-Tree

[7, 5, 1, 8, 3, 6, 0, 9, 4, 2] dizisinin Binary-Search-Tree aşamalarını yazalım.

Rootu 7 aldım. Sağ tarafında büyük sayılar, sol tarafında küçük sayılar olacak şekilde,

             7
            / \
           5    8
          / \    \
         1   6    9
       / \   /    
      0   2  4
           \
            3               şeklinde oluşturulur.
